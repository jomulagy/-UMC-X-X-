### 프로젝트 실행법
-------
1. secrets.json 파일 생성하기
> (1) 노션 개발팀 > 문서 > 3.secrets.json 다운<br>
> (2) Dae_Dong_festival 폴더밑에 추가하기
<br>
2. 가상환경 생성하기
> ```
> python -m venv venv
> ```
<br>

3. 모듈 설치하기
> ```
> pip install -r requirements.txt
> ```
<br>

4. 서버 실행하기
> ```
> python manage.py runserver
> ```
<br>
